# 项目说明
    andorid环境下，编译iconv字符转换库

# android中编译iconv过程
    如果当前环境是ndk-r20版本，且是针对andorid-21以上版本，cpu是arm的，则可以直接使用本项目中的config.h文件，不用重新配制环境
    如果是其它版本，或cpu架构，则需要重新配制交叉编译环境
